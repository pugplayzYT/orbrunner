# Orbrunner

Orbrunner is a 3D game created using LWJGL/Java.

## Features
- 3D rendering with custom shaders
- Modular game engine structure
- In-game UI and HUD
- Real-time simulation and game logic

## Getting Started
To run the game, simply use the Gradle wrapper:
```bash
./gradlew run
```

Or run through an IDE configured with Gradle (e.g., IntelliJ IDEA).

## Dependencies
- LWJGL 3 (Lightweight Java Game Library)
- JOML (Java OpenGL Math Library)
- Gson
