# v3.0 - The Launcher Update
## Release Date: February 26, 2026

The biggest update yet — OrbRunner now has a full game launcher and distribution system.

### What's New
- **Game Launcher** — A sleek, dark-themed JavaFX launcher for installing, updating, and playing OrbRunner
- **Version Management** — Choose any version from the dropdown, switch between them instantly
- **Auto-Update Detection** — The launcher checks for new versions and lets you update with one click
- **Download Progress** — Real-time progress bar showing download speed and file size
- **Version Caching** — Downloaded versions are cached locally so switching back is instant
- **Distribution Server** — Python Flask server for hosting and serving game builds
- **One-Click Upload** — Build and publish new versions with a single command

### Technical
- JavaFX-based launcher with CSS-styled dark cyberpunk theme
- HTTP download with progress tracking via Content-Length headers
- Local version cache at ~/.orbrunner/versions/
- Flask REST API with token-authenticated uploads
- Gradle multi-project setup (game + launcher)
